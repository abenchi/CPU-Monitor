# CPU Monitor

Moniteur CPU en temps réel avec interface web moderne.

![CPU Monitor](https://img.shields.io/badge/Node.js-18+-green)

## Installation rapide

### 1. Installer Node.js (si pas déjà fait)
Télécharge et installe Node.js : https://nodejs.org/ (clique sur le gros bouton vert)

### 2. Télécharger le projet
Télécharge le ZIP : https://github.com/abenchi/CPU-Monitor/archive/refs/heads/main.zip

Puis **extrais le ZIP** (clic droit → Extraire tout)

### 3. Lancer le site

**🍎 Sur Mac :** Double-clique sur `start.command`

**🪟 Sur Windows :** Double-clique sur `start.bat`

Le navigateur s'ouvre automatiquement sur http://localhost:3000

### Alternative (ligne de commande)
```bash
cd CPU-Monitor
npm install
node server.js
```
Puis ouvre : **http://localhost:3000**

---

## Fonctionnalités
- Utilisation CPU en temps réel
- Utilisation RAM
- Usage par cœur avec GHz
- Top 5 des processus
- Historique sur 60 secondes
- Mode sombre/clair
- Mode plein écran
- Mode mini (widget)
- Alerte visuelle si CPU > 80%

## Raccourcis
- **Thème** : Bouton soleil/lune en haut à droite
- **Plein écran** : Bouton en haut à droite
- **Mini mode** : Bouton "Mini" - ouvre une petite fenêtre
